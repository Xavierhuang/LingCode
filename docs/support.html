Frequently Asked Questions

How do I set up my API key?

On first launch, you'll be prompted to enter your API key. You can also configure it later in Settings (‚åò,).

Get your API key from:

  • OpenAI: https://platform.openai.com/api-keys
  • Anthropic: https://console.anthropic.com/settings/keys

Why can't I run terminal commands?

LingCode requires App Sandbox to be disabled for full terminal functionality. This is configured in the entitlements file. If you're still having issues:

  • Ensure you're using a login shell (/bin/zsh -l)
  • Check that your PATH environment variable is set correctly
  • Verify that commands are available in your shell

How do I contribute?

LingCode is open source! Contributions are welcome:

  • Fork the repository
  • Create a feature branch
  • Make your changes
  • Submit a pull request

View Repository

Is my code sent to your servers?

No! All your code stays on your local machine. When you use AI features, your code is sent directly to the AI provider (OpenAI or Anthropic) that you've configured. We don't have any servers that store your code.

Can I use LingCode offline?

Basic editing features work offline, but AI features require an internet connection to communicate with AI providers. You can enable "Local-Only Mode" to ensure no data is transmitted except for AI API calls.

What are the system requirements?

  • macOS 12.0 or later
  • Xcode 14.0 or later (for building from source)
  • API key from OpenAI or Anthropic (for AI features)

How is my data protected?

  • All code and files remain on your local machine
  • API keys are stored securely in macOS Keychain
  • No data is collected or transmitted to our servers
  • AI communication is direct between your app and the AI provider
